Carrierwave is a gem for uploading files. Docsplit is a gem for bursting pdfs into text and images. This project joins them together, lovingly.

Installation

  1. Drop the gem into your Gemfile.

    gem 'carrierwave-docsplit'

Usage

  1. Extend your uploader with the module.

    extend CarrierWave::DocsplitIntegration

  2. Now setup text and or image extraction.

    extract :images => { :to => :thumbs },
        :text => { :to => :text }
